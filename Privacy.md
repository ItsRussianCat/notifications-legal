# Privacy Policy

**Last Updated:** March 1, 2026

## Overview

This privacy policy describes how personal information is collected, used, and protected in connection with the Personal AI Notifications messaging service ("Service").

## Service Description

The Service sends automated SMS notifications from a personal AI assistant system to the account owner's registered phone number. The Service is used exclusively by the account owner for personal, non-commercial purposes.

## Information Collected

The only personal information collected and used by the Service is:

- **Phone number:** The account owner's mobile phone number, provided directly by the account owner during system configuration.

No additional personal information is collected from any users, and no information is collected from third parties.

## How Information Is Used

The phone number is used solely to deliver automated notifications to the account owner, including:

- Task completion alerts
- System status updates
- Reminders and scheduled notifications

## Data Sharing

Personal information is **never** sold, rented, shared with, or disclosed to any third parties for marketing or any other purposes. The phone number is shared only with Twilio, Inc., the third-party messaging provider, solely for the purpose of delivering SMS messages.

## Data Retention

The account owner's phone number is stored in a local system configuration file and retained only as long as the Service is in use. The account owner may delete this information at any time by removing it from the configuration.

## Data Security

Personal information is stored locally on the account owner's personal computing devices and is protected by the security measures of those devices. No personal information is stored on external servers beyond what is required by Twilio for message delivery.

## Message Frequency

Message frequency varies based on system activity. Typical usage ranges from a few messages per day to none. Message and data rates may apply.

## Opt-Out

The account owner may stop receiving messages at any time by:

- Replying **STOP** to any message
- Removing their phone number from the system configuration
- Disabling the notification feature in the application settings

## Contact

For questions about this privacy policy, reply **HELP** to any message received from the Service.

## Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected by updating the "Last Updated" date above.
